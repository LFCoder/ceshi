
一.创建版本库
    1.创建一个空目录用于存放版本的所有文件
    2.git init 命令就会在当前目录下创建了版本库(一个隐藏文件.git)
    3.git add 文件名  用于想版本库中添加文件
    4.git commit 提交文件
二.时光机穿梭
   1.git status 用来查看git版本库中的状态,例如是否有更改却没有体交得文件
   2.git diff   用来查看具体有哪些内容改变了 

三.工作区和暂缓区
   工作区: 就是git版本库所在目录里面所有的文件都是在工作区!
   缓存区: 在版本库中会存有一个缓存区,缓存区存放着上次 git add后所有的内容.
